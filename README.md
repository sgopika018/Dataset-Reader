# Weather Dataset Reader

A simple web app to view weather information for a specific city and date. Weather data is displayed in cards for easy reading.

## Features

- Search weather by **city** and **date**  
- Shows weather results dynamically  
- Displays warning if data is not found  

## How to Use

1. Open `index.html` in your browser  
2. Enter a **city name**  
3. Select a **date**  
4. Click **Show Weather** to see results  

## Technologies

- HTML, CSS, JavaScript  
- Optional: JSON/CSV dataset
